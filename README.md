# 🗄️ Desafios de Banco de Dados — Trilha Back-End

> Este projeto faz parte da minha trilha de estudos focada em **Back-End**. Nele, apresento a modelagem e a resolução de dois desafios de banco de dados (Iniciante e Intermediário) utilizando de forma contínua e integrada o **mesmo banco de dados** (`loja`).

## 🏗️ Estrutura de Arquivos

A organização das pastas foi estruturada para demonstrar o progresso dos commits e os testes que realizei localmente:

```text
Desafios_banco_de_dados/
│
├── Desafio_Iniciante/
│   ├── desafio_iniciante.sql      # Script de criação do banco, tabela inicial e CRUD básico
│   └── prints/                    # Capturas de tela comprovando as queries executadas
│
└── Desafio_Intermediario/
    ├── desafio_intermediario.sql  # Estruturação relacional complementar e queries avançadas
